print()
Для вывода данных на экран используется команда print()
Внутри круглых скобок пишем, что хотим вывести на экран. Если это текст, то обязательно указываем его внутри кавычек. Кавычки могут быть одинарными или двойными. 
print('Мы изучаем язык Python') - внутри скобок аргумент или параметр
Команда print() позволяет указывать несколько аргументов, в таком случае их надо отделять запятыми.
между аргументами команды print() после запятой мы ставим 1 символ пробел
Команда print() записывается только маленькими буквами, другое написание недопустимо, так как в Python строчные и заглавные буквы различны.
Каждая последующая команда print() выводит указанный текст с новой строки.
Команда print() с пустым списком аргументов просто вставляет новую пустую строку.

input()
Для считывания данных в языке Python используется команда input()
name = input() - name - переменная или name = input('Как тебя зовут?')
print('Привет,', name)

Необязательные параметры команды print(): sep, end
print('a', 'b', 'c', sep='*') - добавляет разделитель
a*b*c
print('a', 'b', 'c', end='@') - end вместо перевода строки вставляет символ из кавычек
a b c@d e f@@
print() с пустыми скобками ставит перевод строки
Последовательность символов \n называется управляющей последовательностью и задает перевод строки end='\n'  # перевод строки
Чтобы убрать все дополнительные выводимые символы, можно вызывать командуprint() так: print('a', 'b', 'c', sep='', end='')
print('Python') равно print('Python', end='\n')
Если после вывода данных нужно более одного перевода строки, то необходимо использовать следующий код print('Python', end='\n\n\n')

название переменной всегда должно быть слева от знака равенства
В языке Python можно за одну инструкцию присваивания изменять значение сразу нескольких переменных.
name, surname = 'Timur', 'Guev'
name, surname = input(), input()

name1 = 'Timur'
name2 = 'Gvido'
name1, name2 = name2, name1

Любую строку можно превратить в комментарий, поместив перед ней символ #
